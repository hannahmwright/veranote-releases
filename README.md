# Veranote Releases

Official releases for [Veranote](https://veranote.com) - the AI-powered meeting notes app.

## Download

**[⬇️ Download Latest Version](https://github.com/hannahmwright/veranote-releases/releases/latest)**

### System Requirements
- macOS 10.15 (Catalina) or later
- Apple Silicon (M1/M2/M3) Mac

## Installation

1. Download the `.dmg` file from the [latest release](https://github.com/hannahmwright/veranote-releases/releases/latest)
2. Open the downloaded DMG file
3. Drag **Veranote.app** to your **Applications** folder
4. Launch Veranote from Applications or Spotlight

### First Launch
On first launch, macOS may show a security warning. To open:
1. Right-click (or Control-click) on Veranote in Applications
2. Select "Open"
3. Click "Open" in the dialog

## Auto-Updates

Veranote automatically checks for updates and will notify you when a new version is available. You can:
- **Update Now** - Download and install immediately
- **Update Later** - Dismiss and update when convenient

You can also manually check for updates in **Settings → Updates**.

## Verify Your Download

Each release includes SHA256 checksums. To verify:

```bash
shasum -a 256 ~/Downloads/Veranote_*.dmg
```

Compare the output with the checksum in the release notes.
